## Work

### Imxwallet

Currently i am contributing in an open source project for Immutable X. 
This is a layer-2 scaling solution for NFTs on Ethereum. My task in this project is, 
to build a frontend for the BatchTransfer, where you can transfer your NFTs to Immutable X.
A special feature is, that you can upload an csv-File with all Informations. The data should be 
shown on the frontend and can be modified there. After this all NFTs should transfer at once.

In this Project I am working with React and Typescript.



### LED_Wall
My fellow students and I are currently working on a development project for grad school.

In this project, an LED-Wall is to be controlled with the help of a Raspberry Pi and a camera.
The whole thing should run via a Flask web server.
